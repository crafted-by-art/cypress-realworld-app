This Cypress test suite, "transaction-feeds," verifies filtering and visibility of transaction feed items. Tests cover date and amount range filters, ensuring correct transactions display and revert after clearing filters, with network waits and backend data assertions. It also checks no transactions appear when filters exclude all data, and validates feed item ownership: “mine” feed shows personal, “public” shows contact involvement, “friends” shows only contacts. Key commands: cy.wait, cy.pickDateRange, cy.setTransactionAmountRange, cy.getBySelLike, assertions, visual snapshots. Keywords: transaction, filter, feed, date, amount, personal, contact, visibility, assertion, snapshot.
