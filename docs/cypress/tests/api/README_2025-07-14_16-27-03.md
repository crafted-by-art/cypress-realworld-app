## Usage
To integrate and utilize the test cases contained within this folder in a project, follow these steps:
1. Ensure Cypress is installed in your development environment.
2. Place the `api` folder within the Cypress test directory of your project.
3. Run the Cypress test runner to execute these tests by navigating to your project's root folder and executing:
   ```
   npx cypress open
   ```
   or, for headless execution:
   ```
   npx cypress run
   ```
4. Select the individual test files or run all tests together to verify API functionality.
