Test suite "Okta" validates Okta login via API or UI, user onboarding, and logout flows. It seeds the database before each test. Tests confirm users can log in, complete onboarding (creating a bank account), see onboarding steps, and are logged out correctly—including responsive handling for mobile. Additional tests verify absence of bank accounts or notifications post-login. Key commands: `cy.loginByOktaApi`, `cy.loginByOkta`, `cy.intercept`, assertions on DOM visibility/content, and mobile sidebar interaction. Keywords: Okta, Cypress, login, onboarding, logout, fixture, bank account, notifications, mobile, assertions.
