Test suite "Auth0" verifies Auth0 login, onboarding flow, and logout functionality. Before each test, it seeds the database, intercepts GraphQL calls, logs in via Auth0, and visits the homepage. The main test simulates user onboarding (filling bank details), confirms UI dialogs, checks transaction list visibility, and logs out, adapting for mobile if needed. A secondary test confirms onboarding UI is visible on revisiting, leveraging cached login. Key commands: cy.task, cy.intercept, cy.loginToAuth0, cy.getBySel, cy.location. Keywords: Auth0, login, onboarding, logout, Cypress, session caching, mobile, database seed.
