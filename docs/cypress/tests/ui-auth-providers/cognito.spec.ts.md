This file tests user authentication and onboarding via AWS Cognito. It contains two conditional test suites based on an environment variable: one for programmatic API login and another for UI-based login using `cy.origin()`. Before each test, the database is seeded using the `cy.task('db:seed')` command. The main test case validates a full user flow: logging in, completing a multi-step bank account creation form, and logging out. Key commands include `cy.type` to fill inputs, `cy.intercept` to spy on a GraphQL mutation for account creation, and `cy.click` to navigate. Assertions verify UI content and visibility.
