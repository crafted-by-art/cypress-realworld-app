This file tests AWS Cognito login flows using Cypress, with separate suites for programmatic and cy.origin-based login. Tests seed the database, perform Cognito authentication, simulate onboarding (bank account creation), validate onboarding success, and assert logout correctly redirects to home. Key actions use `cy.loginByCognitoApi`, `cy.loginByCognito`, UI validation, GraphQL intercepts, and responsive checks. No external fixtures are loaded. Keywords: AWS Cognito, Cypress, authentication, onboarding, logout, intercept, GraphQL, programmatic login, UI assertion, responsive.
