Test suite "Google" automates login via Google, onboarding, and logout flows if a Google client ID is configured. It seeds the database, intercepts POST requests to /bankAccounts, and logs in using a custom Google API method. The main test steps through onboarding dialogs, fills out bank account details, submits the form, confirms onboarding completion, views the transaction list, and logs out (handling mobile vs desktop UI differences). Another test asserts onboarding visibility. Keywords: Google login, onboarding, bank account, logout, POST intercept, Cypress, db:seed, UI interaction, selectors, mobile detection.
