This file adds two Cypress commands for Amazon Cognito authentication. `loginByCognitoApi` directly signs in via AWS Amplify, fetches JWT tokens, and manually stores them in localStorage to simulate a logged-in user, then visits the homepage. `loginByCognito` uses Cypress's session API and UI automation on a Cognito-hosted login page, entering credentials and verifying login by checking for onboarding content. Both approaches help automate login in test environments. Keywords: Cognito, Amplify, JWT, Cypress.Commands, login, localStorage, session, authentication, AWS, UI automation. No fixtures are loaded.
