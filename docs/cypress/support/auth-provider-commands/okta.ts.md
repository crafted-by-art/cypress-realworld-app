This code adds two Cypress custom commands for automating Okta authentication in tests. "loginByOktaApi" authenticates a user via Okta API, retrieves tokens, stores them in localStorage, and visits the home page. "loginByOkta" simulates interactive login in an Okta-hosted page using cy.origin, fills in credentials, asserts user presence in the UI, and validates the session. Key actions: API request, token handling, localStorage manipulation, UI interaction, session validation. Keywords: Okta, Cypress, authentication, localStorage, session, API, login, custom command, validation, cy.origin.
